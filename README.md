# Библиотека Requests

<div class="t119__preface t-descr" style="font-size:23px;line-height:1.8;font-weight:400;opacity:0.70;" field="text">Модуль <strong><span data-redactor-tag="span" style="color: rgb(0, 124, 119);">Requests </span></strong>предоставляет возможность управления HTTP-запросами при помощи языка Python. Инструментарий библиотеки широкий и рассчитан на все случаи взаимодействия с web-приложениями. Код, написанный с применением <strong><span data-redactor-tag="span" style="color: rgb(0, 124, 119);">Requests</span></strong>, не является громоздким, легко читается, а функции и методы наглядно настраиваются под специфические нужды. <br> <br>Несмотря на то, что в Python встроен модуль <strong><span data-redactor-tag="span" style="color: rgb(0, 124, 119);">urllib3</span></strong>, обладающий сходным функционалом, практически все применяют <strong><span data-redactor-tag="span" style="color: rgb(0, 124, 119);">Requests</span></strong>, что свидетельствует о его удобстве и простоте. <br></div>
